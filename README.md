## Objetivo
<p>construção de um game de ciclos de tarabalho e exercicios voltado para quem passa muito tempo sentado!</p>
 
 
## Tecnologias 
 
 
* Node.js
* React
* React.js
* Typescript
 
 
## como iniciar o projeto
 
* Instale as dependências:
>    $ npm install ou yarn
* Execute o projeto:
>    $ npx start ou yarn start
